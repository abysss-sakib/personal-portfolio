# Sakib — Portfolio Website

## 🚀 Overview
A modern and fully responsive personal portfolio website built with React.js.

It is designed to showcase personal information, skills, projects, and social links in a clean and minimal layout.

---

## ✨ Features
- Fully responsive design (mobile + desktop)
- Smooth navigation between sections
- Dynamic projects system
- Clean and modern UI
- Easy to customize and maintain

---

## ⚙️ Customization
- Update name from Navbar, Home, and Footer
- Edit About Me section from components
- Add or remove projects from the data file
- Update social media links (Facebook, Instagram, GitHub)

---

## ⚠️ Important Notes
- Do not change UI structure or design unnecessarily
- Only modify content (text, links, projects)
- Keep project files clean and organized
